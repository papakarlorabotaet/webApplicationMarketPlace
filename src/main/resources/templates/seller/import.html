📥 Массовая загрузка товаров (CSV)
Требование 1.4.2
Максимальный размер: 10MB | Формат: UTF-8
Отмена
📋 Формат CSV файла

Файл должен содержать 4 колонки в указанном порядке (разделитель — запятая или точка с запятой):

name,description,price,quantity
Поле Тип Обязательно Пример
name Текст ✅ Да Беспроводные наушники
description Текст ✅ Да Отличный звук, 20 часов работы
price Число ✅ Да 2990.00
quantity Число ✅ Да 50
Важно: Все товары после импорта автоматически отправляются на модерацию.
📝 Пример содержимого файла

Так должен выглядеть ваш CSV файл (первая строка — заголовки): 

name,description,price,quantity
Беспроводные наушники,Отличный звук 20 часов работы,2990.00,50
Умные часы,Фитнес-трекер пульс сон,4990.00,30
Чехол для телефона,Силиконовый ударопрочный,590.00,100
USB-кабель Type-C 1м,Быстрая зарядка 3A,290.00,200
Портативное зарядное 10000mAh,Компактный Power Bank,1490.00,75
💡 Советы по заполнению:
  • Не используйте запятые в числовых полях (2990.00 не 2,990.00)
  • Кодировка файла должна быть UTF-8
  • Можно использовать точку с запятой как разделитель (;)
  • Пустые строки будут пропущены автоматически
